## 1. Project Structure & Data Model

First, let's design our Firestore database structure:

### Collections in Firestore:

**`users`** (collection)
- `userId` (document) - auto-generated by Firebase Auth
  - `email`: string
  - `displayName`: string
  - `createdAt`: timestamp

**`habits`** (collection) 
- `habitId` (document) - auto-generated
  - `userId`: string (references users/userId)
  - `title`: string (e.g., "Drink Water")
  - `description`: string (optional)
  - `color`: string (hex color for UI)
  - `icon`: string (emoji or icon name)
  - `createdAt`: timestamp
  - `targetCount`: number (e.g., 8 glasses per day)

**`completions`** (collection)
- `completionId` (document) - auto-generated
  - `userId`: string
  - `habitId`: string (references habits/habitId)
  - `date`: string (format: "YYYY-MM-DD" for easy querying)
  - `count`: number (how many times completed that day)
  - `completedAt`: timestamp

### Sample Code Snippets:

**1. Habit Model (`models/habit.dart`)**
```dart
class Habit {
  final String? id;
  final String userId;
  final String title;
  final String description;
  final String color;
  final String icon;
  final int targetCount;
  final DateTime createdAt;

  Habit({
    this.id,
    required this.userId,
    required this.title,
    this.description = '',
    required this.color,
    required this.icon,
    this.targetCount = 1,
    required this.createdAt,
  });

  // Convert to Map for Firestore
  Map<String, dynamic> toMap() {
    return {
      'userId': userId,
      'title': title,
      'description': description,
      'color': color,
      'icon': icon,
      'targetCount': targetCount,
      'createdAt': Timestamp.fromDate(createdAt),
    };
  }

  // Create Habit from Firestore document
  factory Habit.fromMap(String id, Map<String, dynamic> map) {
    return Habit(
      id: id,
      userId: map['userId'],
      title: map['title'],
      description: map['description'],
      color: map['color'],
      icon: map['icon'],
      targetCount: map['targetCount'],
      createdAt: (map['createdAt'] as Timestamp).toDate(),
    );
  }
}
```

**2. Database Service (`services/database_service.dart`)**
```dart
class DatabaseService {
  final FirebaseFirestore _firestore = FirebaseFirestore.instance;
  final String userId;

  DatabaseService(this.userId);

  // Add a new habit
  Future<void> addHabit(Habit habit) async {
    await _firestore.collection('habits').add(habit.toMap());
  }

  // Get all habits for current user
  Stream<List<Habit>> get habits {
    return _firestore
        .collection('habits')
        .where('userId', isEqualTo: userId)
        .orderBy('createdAt', descending: true)
        .snapshots()
        .map((snapshot) {
      return snapshot.docs.map((doc) {
        return Habit.fromMap(doc.id, doc.data());
      }).toList();
    });
  }

  // Mark habit as completed for today
  Future<void> completeHabit(String habitId, int count) async {
    final today = DateTime.now();
    final dateString = '${today.year}-${today.month}-${today.day}';
    
    await _firestore.collection('completions').add({
      'userId': userId,
      'habitId': habitId,
      'date': dateString,
      'count': count,
      'completedAt': Timestamp.now(),
    });
  }
}
```
